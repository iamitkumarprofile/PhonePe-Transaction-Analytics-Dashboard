# PhonePe Transaction Analytics Dashboard

## Overview
This project is an interactive Power BI dashboard built using PhonePe transaction data. The dashboard provides insights into transaction volume, transaction value, user behavior, payment success rates, and service performance.

## Dashboard Features
- Total Transactions KPI
- Total Transaction Value KPI
- Unique Users Analysis
- Payment Success Rate Tracking
- Monthly Transaction Trend Analysis
- Age Segment Contribution Analysis
- Service-wise Transaction Value Analysis
- Top Users Analysis
- Weekday vs Weekend Usage Analysis

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Data Modeling
- Microsoft Excel
